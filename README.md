# Munix
The Munix Project <br />
v2.3

## Features
- Terminal
- Compressor
- RAM Controller

## Terminal Commands
`r [FILE]` Read <br />
`w [FILE]` Write <br />
`compress [FILE]` Compress <br />
`decompress [FILE` Decompress <br />
`f [NAME]` Find <br />
`print [MESSAGE]` Echo/Terminal Print <br />
`clone [REPO]` GitHub Repo Cloner <br />
Do `help` or `-h` for more commands.
